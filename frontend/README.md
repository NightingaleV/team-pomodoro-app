# Frontend

## Requirements

- Node.js v10.16.2 or later
- `yarn` (`npm install --global yarn`)

## Installation

```sh
yarn install
```

## Run Local Dev Server

```sh
yarn start
```

Open [http://localhost:3000](http://localhost:3000).

## Production Build

```sh
yarn build
```

## Format Code

```sh
yarn prettier
```
